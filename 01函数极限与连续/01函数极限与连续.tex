\chapter{函数、极限与连续}

本章讨论微积分最基本的概念：函数、极限、连续。

本章要点：
\begin{itemize}
    \item 极限。
    \item 连续。
\end{itemize}

重要定理：
\begin{itemize}
    \item 夹逼定理。
    \item 复合函数的连续性定理。
\end{itemize}

重要极限：
\begin{itemize}
    \item $\underset{x\rightarrow 0}{\lim}\frac{\sin x}{x}=1$。
    \item $\underset{x\rightarrow \infty}{\lim}\left( 1+\frac{1}{x} \right) ^x=e$。
    \item $\underset{x\rightarrow 0}{\lim}\frac{a^x-1}{x}=\ln a$
\end{itemize}
